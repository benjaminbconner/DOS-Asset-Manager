# DOS-Asset-Manager
A retro‑inspired, DOS‑style web application for IT departments to manage computer assets.

Fast, keyboard‑first, and nostalgic — like DOS, but with modern reliability.

---

## ✨ Features

- **Inventory management**: Add, edit, retire, and delete assets
- **Command prompt mode**: Type commands (`list`, `add`, `search owner=alice`) with autocomplete and history
- **Keyboard‑first UI**: Navigate with Tab/Enter/Arrow keys, no mouse required
- **Quick filters**: Search by status, type, owner, model, or location
- **Batch actions**: Retire or reassign multiple assets at once
- **Audit log**: Track changes with timestamps and actors
- **Export/Import**: CSV/JSON for backups and bulk edits
- **Themes**: Green CRT, Amber CRT, or Monochrome
